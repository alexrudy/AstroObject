15. util — Utility functions¶

Provides utilities useful in various places in the module.

static util.getVersion()¶

Returns the version number as either a string or tuple. The version number is retrieved from the “VERSION” file, which should contain just the text for the version and nothing else. When the version is returned as a tuple, each component (level) of the version number is a seperate, integer element of the tuple.

static util.validate_filename(resourcename, extension='.fits')¶

Validates a string as an acceptable filename, stripping path components,etc.

..warning:: This function isn’t very good. I wouldn’t use it in its current state.

static util.update(d, u)¶

A deep update command for dictionaries. This replaces the normal dict.update() command, which does not handle nested dictionaries.

Parameters:
  • d – The original dictionary
  • u – The new valued dictionary
static util.func_lineno(func)¶

Get the line number of a function. First looks for compat_co_firstlineno, then func_code.co_first_lineno.

static util.make_decorator(func)¶

Wraps a test decorator so as to properly replicate metadata of the decorated function, including nose’s additional stuff (namely, setup and teardown).

static util.npArrayInfo(array, name=None)¶

Message describing this array in excruciating detail. Used in debugging arrays where we don’t know what they contain. Returns a message string.

>>> arr = np.array([0,1,2,3,4,5,np.nan])
>>> print npArrayInfo(arr,"Some Array")
Some Array has 7 elements with shape (7,). Range [  nan,  nan] Zeros 1 ( 14%). NaNs 1 ( 14%). 
>>> print npArrayInfo(arr[1:-1],"Some Array")
Some Array has 5 elements with shape (5,). Range [    1,    5]

static functions.BlackBody(wl, T)¶

Return black-body flux as a function of wavelength. Usese constants from Scipy Constants, and expects SI units

static functions.Gaussian(x, mean, stdev, height)¶

Rertun a gaussian at postion x, whith mean, stdev, and height

static functions.get_resolution(wavelengths, matched=True)¶

Return the resolution from a set of wavelengths.

Parameters:
  • wavelengths – an array of wavelengths
  • matched (bool) – return resolution elements aligned with wavelengths (return n object. Otherwise, return n-1 objects, for the resolution at the center of each wavelength bin)
Returns:

An array of resolutions
static functions.conserve_resolution(given_resolution, target_resolution)¶

Test whether a target resolution is less than a given resolution.

Parameters:
  • given_resolution – the resolution to test against
  • target_resolution – the resolution for testing
static functions.cap_resolution(given_resolution, target_resolution)¶

Cap the target resolution so that it does not exceed the given resolution.

Parameters:
  • given_resolution – The resolution to cap at.
  • target_resolution – The output resolution, which will be clipped at the interpolated given_resolution.
static functions.get_resolution_spectrum(minwl, maxwl, resolution)¶

Return a constant resolution spectrum.

Parameters:
  • minwl (float) – The starting wavelength.
  • maxwl (float) – The ending wavelength.
  • resolution (float) – The constant resolution
Returns:

Tuple of (wavelegnths, resolutions)
static functions.Resample(old_wavelengths, flux, new_wavelengths, resolution=None)¶

Gaussian resampling of a spectrum.

Parameters:
  • old_wavelengths (array) – The original wavelength data for resampling.
  • flux (array) – The flux of the spectrum at each wavelength.
  • new_wavelengths (array) – The requested wavelengths.
  • resolution (array) – The requesting resolution (only provided if the requesting resolution should not be determined by the requesting wavelengths.)
