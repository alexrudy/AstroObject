Welcome to AstroObject!¶

AstroObject is a library for Object-Oriented Astronomy. Currently under development at github, it contains tools for object-oriented management of Spectrum Data, Image Data, and Analytic Spectrum Data. More features are planned for the future. To get a sense of the philosophy of this project, read the introduction.

Warning

This module is under development. The version number 0.4 is very low, indicating that much of this module may be unreliable. The behavior in specific instances is not guaranteed nor should it be trusted. For more information about the development of this module, see Installation.
